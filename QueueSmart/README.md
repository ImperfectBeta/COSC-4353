# QueueSmart

QueueSmart is a web application that allows organizations to manage their services and queues and allows customers to view the services and queues and join them.

## Tech Stack

- Svelte 5 & SvelteKit
- TypeScript
- Tailwind CSS
- .NET 10 & C#

## How to Run

### Frontend

```bash
cd QueueSmart
npm install
npm run dev
```

### Backend

```bash
cd QueueSmart.Api
dotnet run
```
